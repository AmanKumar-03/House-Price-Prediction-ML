# House-Price-Prediction-ML

A Machine Learning project that predicts house prices using historical real estate data. This project demonstrates an end-to-end ML workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, feature selection, model training, hyperparameter tuning, and performance evaluation to build an accurate house price prediction model.

## Project Overview

The objective of this project is to develop a robust regression model that estimates house prices based on property features such as location, area, number of bedrooms, bathrooms, and other relevant attributes. Multiple machine learning algorithms are trained and compared to identify the best-performing model.

## Key Features

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Data Preprocessing
* Data Visualization
* Model Training and Comparison
* Hyperparameter Tuning using GridSearchCV
* Performance Evaluation using MAE, RMSE, and R² Score
* Model Serialization with Pickle

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Random Forest
* Xgboost
* Pickle

## Machine Learning Workflow

Dataset → Data Cleaning → EDA → Feature Engineering → Data Preprocessing → Train-Test Split → Model Training → Hyperparameter Tuning → Model Evaluation → House Price Prediction

## Model Evaluation Metrics

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

## Purpose

This project demonstrates the practical application of Machine Learning for real estate price prediction by building an accurate regression model that can estimate house prices based on property characteristics. It highlights the complete machine learning lifecycle, from raw data preprocessing to model deployment, making it suitable for data science portfolios and resume projects.
